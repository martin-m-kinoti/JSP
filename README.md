# Java Web Services

This repo focuses on server-side Java web development with Servlets and JSP.

## BIT3208-Week9-Java-Web-Services

A Student Login System built with Jakarta Servlets, JSP, and JSTL, packaged as a Maven WAR for deployment on Tomcat. It covers:

- Handling form submissions with Servlets (`LoginServlet`, `LogoutServlet`)
- Session management (`HttpSession`) to track logged-in users
- Rendering dynamic content with JSP (`login.jsp`, `welcome.jsp`)
- Safe output escaping with JSTL (`<c:out>`) to prevent XSS
- Building and packaging a Java web app with Maven

See the folder's own [README](BIT3208-Week9-Java-Web-Services/README.MD) for setup and run instructions.
